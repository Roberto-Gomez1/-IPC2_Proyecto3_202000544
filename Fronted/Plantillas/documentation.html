Docuemntacion proyecto 3

En el ambito laboral es indispensable el contar con las herrameintas necesarias para manipulacion de datos para la opcion y recibo de datos, siendo estos divididos por 2 secciones Backend y Frontend. Al mismo tiempo se busca las mejores tecnologias que estan a la disposicion para trabjar, en este caso usando como Frontend Django y Backend Flask. Estás aparte de almacenar a la información es la que presenta de dichos datos para poder generar una salida mediante archivos de extensión XML para presentación de datos

Resumenes segun los requeriemintos necesarios

Segun eneunciado

Se dispone de dos distintos tipo de de resumenes de duenta los cuales son:

•Clasificacion por fecha: se encarga de mostrar los datos por cada fecha.

•Por rangos de fecha: segun un rango de fechas se crea un resumen de datos que son filtrados por fechas deseadas.

•Reporte en PDF: Deberá ser posible realizar un reporte de todas las peticiones anteriormente descritas.
